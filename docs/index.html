Installation

1. Prerequisites

  • Python 3.8+
  • PostgreSQL database (for production)
  • SMTP server or email service (Brevo/SendGrid)

2. Clone Repository

git clone https://github.com/narendrancs89/EmailCampaignManager.git
cd EmailCampaignManager

3. Install Dependencies

pip install -r render_requirements.txt

4. Set Environment Variables

Create a .env file:

DATABASE_URL=sqlite:///email_campaign.db
SESSION_SECRET=your_secret_key_here
BREVO_API_KEY=your_brevo_api_key_optional

5. Initialize Database

python create_admin_user.py

6. Run Application

python -m flask --app main run --host=0.0.0.0 --port=5000

Access at: http://localhost:5000

Deployment Options

Render.com (Recommended)

The project includes a render.yaml configuration for easy deployment.

  1. Connect your GitHub repository to Render
  2. Select "Web Service"
  3. Choose this repository
  4. Set environment variables:
    • DATABASE_URL - PostgreSQL connection string
    • SESSION_SECRET - Random secret key
    • BREVO_API_KEY - Optional, for email verification
  5. Deploy!

GitHub Codespaces

For development and testing:

  1. Open in GitHub Codespaces
  2. Run the Flask application
  3. Access via the provided Codespace URL

Local Development

Perfect for customization:

  1. Follow the Installation steps above
  2. Uses SQLite by default
  3. Perfect for testing and development
