Getting Started

System Requirements

Browser

Chrome, Firefox, Safari or Edge (latest)

Internet

Stable connection required

Account

Admin approval needed

First Time Access

  1. Your admin will send a registration link
  2. Click the link and create your account
  3. Verify your email address
  4. Wait for admin approval
  5. Log in and start creating campaigns!

Step-by-Step Workflows

Workflow 1: Your First Campaign

Step 1️⃣: Create a Segment

  1. Click "Segments" → "New Segment"
  2. Enter segment name (e.g., "Q2 Leads")
  3. Add description (optional)
  4. Click "Create Segment"

Step 2️⃣: Add Contacts

Option A: Upload CSV

email,first_name,last_name
john@example.com,John,Doe
jane@example.com,Jane,Smith

Option B: Add Manually

  1. Click "Add Contact"
  2. Enter email and name
  3. Click "Add"

Step 3️⃣: Create Email Template

  1. Go to Templates → New Template
  2. Enter template name
  3. Click "Design Email"
  4. Use visual editor or HTML source
  5. Add personalization: {{first_name}}
  6. Click "Save Template"

Step 4️⃣: Configure SMTP

Provider Server Port
Gmail smtp.gmail.com 587
Outlook smtp-mail.outlook.com 587
SendGrid smtp.sendgrid.net 587

Step 5️⃣: Create Campaign

  1. Click "Campaigns" → "New Campaign"
  2. Enter campaign name and subject
  3. Select template and segment
  4. Customize sender info (optional)
  5. Send test email first
  6. Choose schedule or send immediately

Step 6️⃣: Monitor Results

  • View in real-time in Analytics
  • Check open rate after 24 hours
  • Check click rate after 48 hours
  • Document insights

Workflow 2: Email Design Tips

Design Best Practices

  • Responsive: Looks good on mobile and desktop
  • Colors: Max 2-3 colors
  • Fonts: Use web-safe fonts (Arial, Verdana)
  • Images: Under 2MB, add alt text
  • Spacing: Generous margins for readability

Workflow 3: Admin Management

  1. Select user from "Manage Users"
  2. Click "Edit Permissions"
  3. Check/uncheck abilities
  4. Click "Save"

Best Practices

Subject Line Optimization

Do This

  • Keep under 50 characters
  • Be clear and specific
  • Use power words
  • Personalize when possible

Avoid This

  • ALL CAPS TEXT
  • Too many special chars !!!!
  • Misleading subjects
  • Vague descriptions

Segmentation Strategy

  • By Behavior: Recent buyers, inactive users
  • By Demographics: Location, age, company size
  • By Engagement: Openers, clickers, non-engagers
  • By Purchase: VIP customers, trial users

Timing & Frequency

Send Times

Best Days: Tuesday - Thursday

Best Times: 9-10 AM or 6-7 PM

Frequency: Max 2x per week

Spacing: At least 3 days between campaigns

Key Metrics To Monitor

Metric Target If Low, Do This
Open Rate 20-30% Improve subject lines
Click Rate 2-5% Better CTA buttons
Bounce Rate <1% Clean your list
Unsubscribe <0.5% Reduce frequency

Troubleshooting

Email Not Sending?

Check These:

  • SMTP credentials correct?
  • Port 587 not blocked?
  • Using app-specific password (Gmail)?
  • Segment has valid emails?

Can't Log In?

  1. Check caps lock
  2. Try email instead of username
  3. Use "Forgot Password"
  4. Ensure account approved by admin

CSV Import Issues?

  • Format: email, first_name, last_name
  • Check for special characters
  • Verify email format is valid
  • Try with smaller test file first

Emails Going to Spam?

  • Improve subject line (no caps)
  • Set up SPF, DKIM, DMARC
  • Clean your email list
  • Build sender reputation gradually

Launch Checklist

Before First Campaign

  • Account approved by admin
  • Email verified
  • Segment created with contacts
  • Email template designed
  • SMTP configured and tested
  • Test email sent successfully

Before Scheduling

  • Subject line is compelling
  • Email looks good on mobile
  • All links tested and working
  • Personalization tags working
  • Unsubscribe link included
  • Send time reasonable

After Campaign Sends

  • Monitor delivery in next hour
  • Check for bounce issues
  • Review open rate at 24 hours
  • Review click rate at 48 hours
  • Document learnings
  • Plan next campaign
