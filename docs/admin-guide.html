Admin Overview

What is an Admin?

An admin user has complete control over the Email Campaign Manager system. Admins can:

  • Create and manage user accounts
  • Approve or reject new user registrations
  • Set permissions for individual users
  • Create system-wide email lists
  • View comprehensive analytics
  • Manage SMTP configurations

Key Responsibilities

User Approval

Review and approve new user registrations

Permission Management

Control what each user can do

System Monitoring

Track overall system health and usage

Maintenance

Handle user issues and troubleshooting

User Management

Viewing All Users

  1. Click "Admin" in the main menu
  2. Click "Manage Users" or "Users"
  3. View the list of all users with their details:
    • Username and email
    • Account status (active/inactive)
    • Admin status
    • Last login date
    • Creation date

Create a New User Manually

  1. Click "Admin" → "Create User"
  2. Fill in user details:
    • Username (must be unique)
    • Email (must be unique)
    • Password (generate secure password)
    • Is Admin? (checkbox)
  3. Click "Create"
  4. System will send welcome email to user

⚠️ New User Best Practices

  • Use strong, unique usernames
  • Generate strong passwords (12+ characters, mixed case, numbers, symbols)
  • Send welcome email with temporary password
  • Ask user to change password on first login

Edit User Details

  1. Find user in the list
  2. Click the user's name or "Edit" button
  3. Modify:
    • Email address
    • Admin status
    • Account status
  4. Click "Save Changes"

Deactivate a User

  1. Find user in the list
  2. Click "Deactivate" button
  3. Confirm the action
  4. User will be unable to log in
  5. Their campaigns will be paused

Reactivate a User

  1. Find the deactivated user (may be in separate list)
  2. Click "Activate" button
  3. User can log in again

Delete a User (Permanent)

⚠️ WARNING: This is irreversible!

  1. Find user in list
  2. Click "Delete" button
  3. Confirm twice (this action cannot be undone)
  4. User and all their data will be permanently removed

Manage Registration Requests

Review Pending Registrations

  1. Click "Admin" → "User Requests"
  2. View pending registration requests with:
    • Requested username
    • Email address
    • Request date
    • Request reason/notes

Approve a Registration

  1. Click request to view details
  2. Review the information
  3. Click "Approve" button
  4. Choose an action:
    • Create account for them
    • Send approval email only
  5. User receives approval notification
  6. User can now log in

Reject a Registration

  1. Click request to view details
  2. Click "Reject" button
  3. Enter reason for rejection (optional)
  4. Click "Send Rejection"
  5. User receives rejection email with reason

Permission Management

Available Permissions

Permission Description Default
can_create_campaigns Create and schedule email campaigns ✅ Yes
can_manage_templates Create, edit, delete templates ✅ Yes
can_manage_segments Create and manage email segments ✅ Yes
can_import_contacts Upload contacts via CSV ✅ Yes
can_view_analytics View campaign analytics and reports ✅ Yes
can_manage_smtp Configure SMTP settings ✅ Yes

Set User Permissions

  1. Go to "Manage Users"
  2. Click on a user's name
  3. Click "Edit Permissions"
  4. Check/uncheck permissions as needed
  5. Click "Save Permissions"
  6. Changes take effect immediately

Permission Presets

👤 Standard User

✅ All campaign and template features

❌ Cannot manage other users or global settings

🔓 Limited User

✅ Can only view analytics and create campaigns

❌ Cannot upload new contacts or create templates

👩‍💼 Manager

✅ All standard features + admin capabilities

❌ Cannot delete users (only main admin)

Reset User Password

  1. Find user in the management list
  2. Click "Reset Password"
  3. Generate a temporary password
  4. Send password to user securely
  5. Ask user to change password on next login

Admin Email Lists

What are Admin Email Lists?

Admin Email Lists are system-wide contact lists managed by admins. Unlike user segments, these lists:

  • Can be shared across multiple users
  • Are managed centrally by admins
  • Ideal for shared contacts (customers, partners)
  • Can have controlled access levels

Create Admin Email List

  1. Click "Admin" → "Email Lists"
  2. Click "Create New List"
  3. Enter list details:
    • List name
    • Description
    • Access level (private/shared)
  4. Click "Create"

Add Contacts to List

Option A: Import CSV

  1. Click on the list name
  2. Click "Import Contacts"
  3. Upload CSV file

Option B: Add Manually

  1. Click on the list
  2. Click "Add Contact"
  3. Enter email and details

Manage List Access

  1. Select the list
  2. Click "Edit"
  3. Set access level:
    • Private: Only you can use
    • Shared: All users can use
    • Restricted: Specific users only
  4. Save changes

Security & Best Practices

Admin Account Security

🔒 Protect Your Admin Account

  • Use a strong password (20+ characters recommended)
  • Enable two-factor authentication if available
  • Never share your account credentials
  • Log out after each administration session
  • Monitor login activity regularly

User Data Protection

  • Never share user emails or personal information
  • Deactivate accounts of inactive users
  • Regularly audit user permissions
  • Delete accounts of terminated employees
  • Keep audit logs of admin actions

SMTP Security

⚠️ SMTP Best Practices

  • Use TLS encryption (port 587)
  • Never use plain text passwords
  • Use app-specific passwords
  • Rotate SMTP credentials annually
  • Monitor SMTP usage for anomalies

Regular Maintenance

Task Frequency Purpose
Review user accounts Monthly Remove inactive users
Audit permissions Quarterly Ensure correct access levels
Database backup Weekly Data recovery protection
Check system logs Daily Detect issues early

Troubleshooting Admin Issues

User Cannot Log In

Check: Is account active? Email verified? Account approved? Password correct?

Emails Not Sending

Check: SMTP configuration correct? Credentials valid? Port not blocked?

High Bounce Rate

Check: Email list quality? Are addresses valid? Check for typos?
